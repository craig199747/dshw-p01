二、数据来源
所有数据均为纯线上获取，无任何手动输入：
股票行情：baostock，后复权价格，日度频率，时间范围 2021-01-01 至 2026-05-20
市场指数：沪深 300（000300）、中证 500（000905），来源 baostock；选择理由：沪深 300 代表大盘蓝筹，中证 500 代表中小盘成长，可对比不同市值风格的市场表现
宏观指标：CPI 同比增速、M2 同比增速，来源 akshare；选择理由：CPI 反映通货膨胀水平，M2 反映市场流动性，两者是影响股市走势的核心宏观因素
财务数据：ROE、净利润率，来源 baostock 杜邦分析接口；ROE 直接获取，净利润率通过杜邦公式（ROE/(权益乘数 × 资产周转率)）计算
三、存储方式
3.1 基础存储：CSV 格式（方式 A，必做）
所有原始数据和清洗后数据均以 CSV 格式存储，目录结构如下：
plaintext
data/
├─ stock/          # 原始个股行情数据（10个CSV）
├─ index/          # 原始指数数据（2个CSV）
├─ macro/          # 原始宏观数据（2个CSV）
├─ finance/        # 原始财务数据（1个CSV）
├─ clean/          # 清洗后数据
│  └─ stock_clean.csv
└─ combined/       # 合并后综合数据
   └─ combined_data.csv
CSV 格式优缺点
优点：
通用性极强，几乎所有数据分析软件和编程语言都支持
纯文本格式，可读性好，可直接用记事本打开查看和编辑
无需额外依赖库，使用简单，学习成本低
力不从心的场景：
数据量达到百万行以上时，文件体积急剧膨胀，占用大量磁盘空间
读取速度慢，需要解析文本并推断数据类型，大数据集下性能差
不支持列式查询，必须读取全部内容才能获取部分列
无数据类型约束，容易出现类型错误（如股票代码前导零丢失）
不支持复杂数据结构和嵌套数据
3.2 进阶存储：Parquet 格式（方式 B，选做）
在 CSV 基础上，将清洗后的股票数据额外保存为 Parquet 格式：
plaintext
data/clean/
├─ stock_clean.csv         # 基础格式
└─ stock_clean.parquet     # 进阶格式
选择 Parquet 的理由
极高的压缩比：本次数据中，Parquet 文件体积仅为 CSV 的 1/5 左右，大幅节省磁盘空间
快速读取：读取速度比 CSV 快 2-3 倍，且支持列式查询，可只加载需要的列
自带 Schema：存储了完整的数据类型信息，无需解析推断，避免类型错误
跨平台跨语言：是大数据分析领域的标准格式，支持 Python、Java、Spark 等多种语言
实现简单：相比 SQLite，无需编写 SQL 语句，代码实现更简洁，适合本次作业的数据分析场景
与 CSV 的性能对比（本次数据规模）
表格
格式	读取耗时	文件大小
CSV	0.012s	1245 KB
Parquet	0.004s	236 KB
差异说明：本次数据规模（约 1.3 万行）下差异已较为明显；当数据量达到百万行以上，或只需要读取部分列时，Parquet 的优势会呈指数级放大。

六、如何运行
1. 安装依赖
bash
运行

2. 下载原始数据
运行01_download.ipynb，自动下载所有个股、指数、宏观和财务数据，保存至data/对应目录。
3. 数据清洗与存储
运行02_clean.ipynb，完成数据清洗、缺失值处理、异常值标注，并保存为 CSV 和 Parquet 格式，同时演示两种格式的性能对比。
4. 数据分析与可视化
运行03_analysis.ipynb，生成基本统计量、5 张可视化图表和 CAPM 模型分析结果。
5. 生成最终报告
在项目根目录运行以下命令，生成独立的 HTML 分析报告：
bash
运行
jupyter nbconvert --to html 03_analysis.ipynb --output report.html --no-input --embed-images --template lab

八、注意事项
路径问题：所有代码使用os.getcwd()自动获取项目根目录，确保在dshw-p01文件夹内运行 Notebook
依赖版本：pandas 版本需≥2.2.0，否则resample("ME")会报错
股票代码格式：已在清洗阶段将股票代码转换为 6 位字符串，避免前导零丢失问题
API 限流：数据下载时添加了 1-2 秒的延时，避免触发服务器限流
报告生成：生成 HTML 报告时使用--embed-images参数，所有图片会嵌入到 HTML 文件中，可独立打开阅读
